    \section{1.4 Separable Equations and
Applications}\label{separable-equations-and-applications}

The equation \[
\frac{dy}{dx} = H(x,y)
\] is called \textbf{separable} if \(H(x,y)\) can be written as the
product of a function of \(x\) and a function of \(y\): \[
\frac{dy}{dx} = g(x)h(y) = \frac{g(x)}{f(y)}
\]

In this case the variables can be isolated \emph{separated}.

    \subsection{Example}\label{example}

Solve the initial value problem \[
\frac{dy}{dx} = -6xy, \qquad y(0)=7
\]

Solution:

\[
\frac{dy}{y} = -6x*dx\]
\[\int\frac{dy}{y} =\int-6x*dx\]
Since the initial value is positive, we can declare y positive inside the ln
\[\ln(y)=-3x^2+C\]
\[y=e^{(-3x^2+C)}\]

Find the value of C through the initial value

\[7=e^{(-3(0)^2+c)}\]
\[7=e^c\]
\[\ln(7)=c\]
\[y=e^{(-3x^2+\ln(7))}\]

    \subsection{Implicit, General, Singular
Solutions}\label{implicit-general-singular-solutions}

The equation \[
K(x,y)=0
\] is called an \textbf{implicit solution} of DE if it is satisfied (on
some interval) by some solution \(y=y(x)\) of the DE.

\subsubsection{Note}\label{note}

\begin{itemize}
\tightlist
\item
  A particular solution \(y=y(x)\) of \(K(x,y)=0\) mau or may not
  satisfy the givent IVP. For example \(x^2+y^2=4\) gives \[
  x+y\frac{dy}{dx}=0
  \]
\item
  Do not assume that every possible algebraic solution \(y=y(x)\) of an
  implicit solution satisfies the same DE.
\item
  Also, solutions of a given DE can be either gained or lost when it is
  multiplied or divided by an algebraic factor. \[
  (y-2x)y\frac{dy}{dx}=-x(y-2x)
  \] \textbf{extraneous solutions}
\end{itemize}

    \subsubsection{General and Particular
Solution}\label{general-and-particular-solution}

A solution that contains an "arbitrary constant" is called a
\textbf{general solution}. A solution with a specific value of the the
constant is called a \textbf{particular solution}.

\subsubsection{Singular Solution}\label{singular-solution}

A particular solution that cannot be obtained from the general solution.
(ex. \((y')^2=4y\) has general solution \(y=(x-C)^2\), but \(y=0\) is a
singular solution)

\subsection{Example}\label{example}

Solve the DE \[
\frac{dy}{dx}=6x(y-1)^{2\over 3}
\]
Solution
\[\frac{(y-1)^{\frac{2}{3}}}{dy}=6xdx\]
\[\int\frac{dy}{3(y-1)^{\frac{2}{3}}}=\int2xdx\]
\[(y-1)^{\frac{1}{3}}=x^2+C\]
\[y=(x^2+C)^3+1\]
y=1 is a singular solution.
